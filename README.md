# MVP Sistema de Inventario

**Versión:** 1.0.0

## Descripción

Este es un proyecto simple que explora las capacidades de **Spring Boot** mediante la creación de una **API REST** como backend para gestionar operaciones **CRUD** en una base de datos local.

El proyecto utiliza una tabla llamada `productos` que es autogenerada por Spring Boot, y genera automáticamente los métodos de CRUD y los endpoints de la API.

## Tecnologías

- **Java** - Lenguaje de programación
- **Spring Boot** - Framework backend
- **HTML y JavaScript** - Frontend
- **Base de datos local** - Almacenamiento

## Clonar el Repositorio

```bash
git clone https://github.com/mavasquez01/MVP_sistema_inventario.git
cd MVP_sistema_inventario
```

## Requisitos

- Java 11 o superior
- Maven 3.6 o superior

## Ejecutar la Aplicación

```bash
mvn spring-boot:run
```

La aplicación estará disponible en: `http://localhost:8080`
