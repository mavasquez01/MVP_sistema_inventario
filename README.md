# MVP_sistema_inventario
V.1.0
  Este es un projecto simple buscando explorar las capacidades de Spring Boot mediante la creación de una API Rest como backend para gestionar CRUD hacia una base de datos local.
  Se utiliza una sola tabla en la base de datos llamada productos la cual es autogenerada mediante Spring Boot, a su vez, se generan los metodos de CRUD y los Endpoints de la API.
